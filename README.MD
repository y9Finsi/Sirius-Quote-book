## Установка и запуск

### Клонировать репозиторий:
```bash
git clone https://github.com/y9Finsi/Sirius-Quote-book
```

### Выдать права доступа скрипту:
```bash
chmod +x main.sh
```

### Запустить скрипт:
```bash
./main.sh
```
